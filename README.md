# Multi-Page Responsive Website

A professional and fully responsive multi-page website built using **HTML, CSS, and JavaScript**.  
This project demonstrates clean UI design, responsive layouts, page navigation, and consistent styling across multiple pages.

---

# Objective

The objective of this project is to build a complete multi-page website structure with responsive design and smooth navigation between pages.

---

# Features

- Multi-page website structure
- Responsive design for mobile and desktop
- Navigation bar with active page highlighting
- Mobile hamburger menu
- Home page hero section
- About page
- Services section with cards
- Contact form
- Consistent UI across all pages
- Clean and modern design
